# logic.js
## VERSÃO PORTUGUESA:
Este mini projecto é uma calculadora básica de números aleatórios. Usei a lógica matemática, os operadores aritmétrico e o objeto Math.
O objeto Math.random produz números de ponto flutuantes maiores que zero e menores que um, no entanto multiplique o Math.random com o número
10 e encapsulei num metodo 'Math.floor' que aredonda o resultado para um numero menor proximo, criando assim números inteiros aleatórios de 0 a 10 e realizando os calculos básico. Este é o meu primeiro projeto.

## ENGLISH VERSION:
This is my first JavaScript project. It utilizes the Math object to generate random integers between 0 and 10 and performs basic arithmetic operations. It handles edge cases such as division by zero and square roots of negative numbers, while formatting decimal outputs for better readability.

# Key Skills:
* Logic & Conditional Statements
* JavaScript Math Object (random, floor, sqrt, cbrt)
* String Interpolation (Template Literals)
